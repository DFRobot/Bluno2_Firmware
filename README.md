## DFRobot_Bluno2
在组网模式下，传输的数据包需要编码。这是一个编码库，用于处理蓝牙事件，编码蓝牙数据的发送和接收。详细信息请阅读这个库的README文件

## 烧录方法（补充图片）
* 不接通电源, 按下BOOT按键不放
* 插上USB电源
* FireBeetle Board-328P with BLE4.1进入双灯闪烁模式
* 软件载入固件.img
* 单击下载按键
